# Checklist Rilis Google Play — NarVara BrainGame Package: `com.narvara.braingame` · Org EAS: `narvaradev` · Project: `@narvaradev/narvara-braingame` --- ## ✅ FASE 1 — Build (sedang/selesai) - [x] Environment variables di EAS (production + preview) - [x] `TEST_DEVICE_IDS` hanya aktif saat dev - [x] AdMob App ID benar (`~2479229803`), interstitial benar (`/3475101334`) - [ ] Build AAB selesai → download `.aab` dari link EAS --- ## 📄 FASE 2 — Privacy Policy (WAJIB sebelum publish) Play butuh **URL publik** kebijakan privasi. Pakai file `PRIVACY_POLICY.md` yang sudah dibuat. **Cara hosting gratis via GitHub Pages:** 1. Repo: `NarVara-Development/BrainGameAndroid` → tab **Settings → Pages**. 2. Source: branch `main`, folder `/ (root)` → Save. 3. URL jadi: `https://narvara-development.github.io/BrainGameAndroid/PRIVACY_POLICY` (atau buat `docs/index.html`). **Alternatif paling cepat:** buka file `PRIVACY_POLICY.md` di GitHub → klik **Raw**, atau pakai layanan seperti https://www.freeprivacypolicy.com. URL ini dipakai di Play Console **dan** di form Data Safety. --- ## 🏪 FASE 3 — Buat App di Play Console 1. https://play.google.com/console → **Create app**. 2. App name: **NarVara BrainGame**, Default language: **Indonesia**, Type: **Game**, Free. 3. Setujui declarations. --- ## 🛒 FASE 4 — Produk In-App: `narvara_braingame_premium` > ⚠️ Produk in-app **hanya bisa dibuat setelah AAB pertama diupload** ke salah satu track (Internal testing cukup). 1. Upload AAB dulu (Fase 6). 2. **Monetize → Products → In-app products → Create product**. 3. Isi: - **Product ID:** `narvara_braingame_premium` ← HARUS sama persis dengan `PREMIUM_SKU` di kode (`stores/purchaseStore.ts`). - **Name:** Premium — Hapus Iklan - **Description:** Hapus semua iklan selamanya. - **Type:** One-time product (non-consumable / managed). - **Price:** mis. Rp 25.000 (atur sesukamu). 4. **Activate** produk. --- ## 🔒 FASE 5 — Form Wajib di Play Console - [ ] **Privacy Policy URL** (App content → Privacy policy) - [ ] **Data Safety**: deklarasikan data yang dikumpulkan: - Email & nama (akun) — dikumpulkan, untuk fungsi akun, dapat dihapus atas permintaan. - Skor/aktivitas dalam app — analytics/fungsi app. - Advertising ID — untuk iklan (AdMob). - In-app purchase — tidak menyimpan data kartu (Google yang proses). - Semua dienkripsi saat transit; ada cara minta hapus data (email). - [ ] **Ads**: pilih **Yes, app contains ads**. - [ ] **Content rating**: isi kuesioner IARC (ekspektasi Everyone/PEGI 3). - [ ] **Target audience**: 13+ (atau sesuai). - [ ] **Government app**: No. - [ ] **News app**: No. - [ ] **Data deletion**: cantumkan email penghapusan. --- ## 📦 FASE 6 — Upload AAB & Rilis 1. **Testing → Internal testing → Create new release**. 2. Upload file `.aab` (dari EAS). 3. Isi **Release notes** (mis. "Rilis perdana — 10 game asah otak"). 4. Tambahkan email tester (akun Gmail-mu) di **Testers**. 5. **Review release → Start rollout to Internal testing**. 6. Install lewat link tester untuk verifikasi. > Setelah lolos internal testing → naik ke **Closed/Open testing** → **Production**. ### Alternatif submit via EAS (otomatis) Setelah app dibuat & ada service account JSON: ```powershell npx eas-cli submit -p android --profile production ``` (butuh `play-service-account.json` — JANGAN commit; sudah di-gitignore.) --- ## 🎨 FASE 7 — Aset Grafis (lihat STORE_LISTING.md) - [ ] Feature graphic 1024×500 - [ ] Min 2 screenshot HP (1080×1920) - [ ] Icon 512×512 --- ## 🚀 Ringkasan urutan ``` Build AAB → Hosting Privacy Policy → Create app → Upload AAB (internal) → Buat produk in-app → Isi Data Safety/Rating/Ads → Rollout internal → Production ```
